# Automated Crack Detection in Concrete Structures Using Image Classification

## Project Overview
This project is developed for the course **Introduction to Computer Vision in Quality Control**. The goal is to build a computer vision system that classifies images of concrete surfaces into two categories: **crack** and **non-crack**.

## Problem
Manual crack inspection in concrete structures is time-consuming and may be inconsistent due to human fatigue, lighting conditions, and subjective judgment. This project aims to support quality control by automating the initial screening process.

## Dataset
We use the **Concrete Crack Images for Classification** dataset:
https://data.mendeley.com/datasets/5y9wdsg2zt/2

The dataset contains 40,000 RGB images of concrete surfaces divided into two classes:
- Crack
- Non-crack

## Planned Approach
The planned solution uses a **Convolutional Neural Network (CNN)** for binary image classification. The pipeline includes:
- dataset loading
- preprocessing
- training and validation
- evaluation using accuracy, recall, precision, and F1-score
