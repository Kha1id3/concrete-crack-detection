# Data

This folder is reserved for raw and processed datasets used in the project.

Raw dataset files should be downloaded and stored locally in:

- `data/raw/concrete_crack_images/`
- `data/raw/sdnet2018/`

Processed datasets can be saved in:

- `data/processed/concrete_crack_images/`
- `data/processed/sdnet2018/`

The datasets are not pushed to GitHub because they are large. Only the folder structure is tracked with `.gitkeep` files.
